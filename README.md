# Boat-Report
## 📊 boAt Business Intelligence Dashboard:-

The boAt Sales Analytics Dashboard is an end-to-end Business Intelligence solution developed using Power BI. It provides a comprehensive analysis of sales performance, customer behavior, and order management by transforming raw business data into meaningful insights through interactive dashboards and KPIs.

The dashboard helps business users monitor revenue, customer engagement, product performance, and return operations, enabling informed and data-driven decision-making.

## 📄 Description &  🎯 Purpose:-
An end-to-end Power BI dashboard that analyzes ₹125.38M in revenue, 22K orders, 12K customers, and 89K+ products sold using interactive dashboards, DAX measures, and business intelligence reporting.
To help businesses monitor key performance metrics, analyze customer behavior, optimize order management, and support data-driven decision-making through interactive analytics.


## 🛠️ Tech Stack :-
- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Microsoft Excel

## 📂 Data Source

The dashboard is built using a **boAt sales dataset** in **Microsoft Excel**, containing transactional data related to sales, customers, products, orders, returns, payment methods, delivery status, and customer demographics. This dataset is used to analyze business performance, customer behavior, product trends, and order operations through interactive Power BI dashboards.


## 🎯 Business Problem

Businesses often struggle to monitor sales performance, understand customer purchasing behavior, and identify operational issues such as returns and cancellations. Without centralized reporting, decision-making becomes slower and less effective.
This dashboard addresses these challenges by providing an interactive view of key business metrics across sales, customers, and order operations.

## 📈 Business Insights

- Generated **₹125.38M** in total sales revenue.
- Successfully processed **22K** customer orders.
- Served over **12K** unique customers.
- Sold more than **89K** products across multiple categories.
- Amazon contributed the highest order volume.
- Identified major return reasons to improve product quality.
- Measured a **20.45%** return rate.
- Evaluated customer loyalty through new vs repeat customer analysis.
